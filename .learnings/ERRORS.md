# OpenClaw Errors

Command failures and integration errors captured during operations.

---

## Error Log Format
```
### [YYYY-MM-DD] <error_type> — <component>
- **Command**: <command that failed>
- **Error**: <error message>
- **Root Cause**: <diagnosis>
- **Fix Applied**: <what was done>
- **Prevention**: <how to avoid in future>
```
