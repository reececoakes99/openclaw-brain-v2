# ISO8583 Message Specification

## Overview

ISO8583 is the dominant financial messaging protocol for card payments. Versions: 1987, 1993, 2003. Most switches use the 1993 variant.

## Message Structure

```
┌──────┬──────────┬──────────┬─────────────────────────────┬──────┐
│ MTI  │ Bitmap    │ DE Data  │ Secondary Bitmap (optional) │ STX  │
│  4   │   8/16    │ Variable │           8                  │  1   │
└──────┴──────────┴──────────┴─────────────────────────────┴──────┘
```

### Field Types

| Type | Format | Example |
|------|--------|---------|
| Fixed Numeric (N) | Right-padded zeros | `000123` |
| Fixed Alpha (AN) | Space-padded | `JOHNDOE` |
| Variable LLVAR | 2-digit len + value | `06JOHNDO` |
| Variable LLVARb | 1-digit len + value | `6JOHNDOE` |
| Variable LLLVAR | 3-digit len + value | `026JOHNDOE` |

## MTI (Message Type Identifier)

| MTI | Name | Direction | Description |
|-----|------|-----------|-------------|
| `0100` | Auth Request | Acquirer → Switch | Card authorization |
| `0110` | Auth Response | Switch → Acquirer | Auth result |
| `0120` | Auth Repeat Request | Acquirer → Switch | Repeat auth |
| `0121` | Financial Request | Acquirer → Switch | Capture/settlement |
| `0130` | Reversal Request | Both | Full/partial reversal |
| `0400` | Network Management | Both | Sign-on/off, echo |
| `0800` | Echo / Test | Both | Ping/pong |

## Processing Code (DE 3) — Key Codes

```
First 2 digits (Transaction Type):
  00 = Purchase / Goods
  01 = Cash
  20 = Refund / Credit
  30 = Deposit
  40 = Transfer

Next 2 digits (From Account):
  00 = Default
  10 = Savings
  20 = Checking

Next 2 digits (To Account):
  00 = Default
  40 = Credit
```

## Building a Message

```python
from dataclasses import dataclass
from typing import Optional
import struct

class ISO8583Message:
    def __init__(self, mti: str, primary_bitmap: int, de: dict):
        self.mti = mti
        self.primary_bitmap = primary_bitmap  # 8 bytes
        self.de = de

    def build(self) -> bytes:
        # Bitmap as hex string
        bitmap = format(self.primary_bitmap, '016X')
        fields = []
        for bit in range(1, 65):
            if self.primary_bitmap >> (64 - bit) & 1:
                fields.append(self.pack_field(bit, self.de.get(bit)))

        data = ''.join(fields)
        raw = self.mti + bitmap + data

        # Calculate MAC (ISO9797-1 block cipher)
        mac = self.calculate_mac(raw.encode('ascii'))
        return raw.encode('ascii') + mac

    def pack_field(self, de_num: int, value: str) -> str:
        if de_num in [2, 14, 35, 36]:  # LL
            return f"{len(value):02d}{value}"
        if de_num in [48, 62, 63]:  # LLL
            return f"{len(value):03d}{value}"
        return str(value).ljust(self.field_length(de_num), ' ')[:self.field_length(de_num)]

    @staticmethod
    def field_length(de_num: int) -> int:
        lengths = {
            2: 19, 3: 6, 4: 12, 7: 10, 11: 6, 12: 6, 13: 4,
            14: 4, 15: 4, 16: 3, 18: 4, 22: 3, 25: 2,
            32: 11, 33: 11, 37: 12, 38: 6, 39: 2,
            41: 8, 42: 15, 43: 40, 49: 3, 52: 8,
            53: 16, 55: 512, 57: 6, 62: 12, 63: 10,
        }
        return lengths.get(de_num, 0)


# Example: Build an authorization request
msg = ISO8583Message(
    mti="0100",
    primary_bitmap=(
        (1 << (64-2)) |   # DE2 PAN

        (1 << (64-7)) |   # DE7 Date/Time

        (1 << (64-13)) |  # DE13 Local Date
        (1 << (64-14)) |  # DE14 Expiry
        (1 << (64-22)) |  # DE22 POS Entry Mode
        (1 << (64-25)) |  # DE25 Condition Code
        (1 << (64-37)) |  # DE37 RRN
        (1 << (64-39)) |  # DE39
        (1 << (64-41)) |  # DE41 TID
        (1 << (64-42)) |  # DE42 MID

    ),
    de={
        2: "4111111111111111",
        3: "000000",  # Purchase, default account
        4: "0000001000",  # EUR 10.00

    }

```

## Parsing a Message

```python
def parse(raw: bytes) -> ISO8583Message:
    mti = raw[:4].decode()
    bitmap_bytes = raw[4:12]
    primary_bitmap = int.from_bytes(bitmap_bytes, 'big')

    pos = 12
    de = {}
    for bit in range(1, 65):
        if primary_bitmap >> (64 - bit) & 1:
            value, pos = unpack_field(raw, bit, pos)
            de[bit] = value

    return ISO8583Message(mti, primary_bitmap, de)
```

## MAC Calculation (ISO9797-1)

```python
from Crypto.Cipher import DES3

def calculate_mac(data: bytes, key: bytes) -> bytes:
    # ISO9797-1 Algorithm 3 (CBC)
    cipher = DES3.new(key, DES3.MODE_CBC, iv=b'\x00' * 8)
    padded = data + b'\x80' + (7 - (len(data) + 1) % 8) * b'\x00'
    return cipher.encrypt(padded)[-8:]
```

## ISO8583 Test Checklist

- [ ] Malformed bitmap (bit flip, odd number of hex chars)
- [ ] Overlapping field boundaries (LLVAR overflow)
- [ ] Replay attack — same STAN + date
- [ ] Invalid MTI combinations (e.g., 0200 response)
- [ ] DE length violations (underflow/overflow)
- [ ] Missing required fields for MTI
- [ ] Invalid PAN (non-numeric, bad Luhn)
- [ ] Negative or zero amount
- [ ] Expired card (DE14 < current date)
- [ ] Missing MAC (if required by processor)
- [ ] Invalid MAC
- [ ] Timeout (no response after 30s)
